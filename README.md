#ECサイト＆Marketplace

React、Redux, Node、Express、およびPostgresを使用して開発。Stripe APIを使用して商品を販売。製品検索と提案、ショッピングカート、注文管理、支払い処理、配送管理、レビュー管理、を含むオンライン市場アプリケーションの開発。

### [本番DEMO](http://https://safe-brushlands-47729.herokuapp.com/")
